Research and Projects

Here are some notable academic projects and research works of mine:

  • Significant improvement of fidelity for encoded quantum bell pairs at long and short-distance communication along with generalized circuit

    Accepted for publication in Heliyon. DOI: 10.1016/j.heliyon.2023.e19700 [quant-ph]

    arXiv preprint link: arXiv:2303.07425v1 [quant-ph]

  • Effect of Quantum Repetition Code on Fidelity of Bell States in Bit Flip Channels

    This paper is based on my undergraduate thesis, supervised by Dr. Md. Saifur Rahman, Professor, Dept. of EEE, BUET. Accepted at the 12th International Conference on Electrical and Computer Engineering (ICECE). DOI link: https://doi.org/10.1109/ICECE57408.2022.10088665.

  • Effect of EOG Artifact Removal on EEG Motor-Imagery Classification

    This paper is based on our Biomedical Engineering sessional course project during L4T2. Accepted at the 12th International Conference on Electrical and Computer Engineering (ICECE). DOI link: https://doi.org/10.1109/ICCIT57492.2022.10056062.

  • Image Encoding and Decoding using FRQI and NEQR Method using Qiskit

    This was an assignment project during my RA-ship. Link to Jupyter notebook: here.

  • Load Flow Analysis for n Bus using Gauss-Siedel Algorithm

    It was our sessional course project for Power System II during L3T2. I utilized Microsoft Excel for input and output data and MATLAB to generalize the algorithm and generate the output. GitHub Link: https://github.com/syedshubha/load_flow_analysis_gauss_seidel

  • Image Encryption using XOR operation and Random Permutation Generation from password:

    This was our Numerical Techniques Sessional Course project during L2T1. MATLAB was used for this project. GitHub Link: https://github.com/syedshubha/ImageEncryption

  • Bangla Speech Emotion Detection:

    Using Machine Learning, four Bengali speech emotions—Happy, Sad, Angry, and Neutral—were classified. This was our course project for DSP-I during L3T1.
